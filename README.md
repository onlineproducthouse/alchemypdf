# AlcheMyPDF

This repository functions as a reference to the AlcheMyPDF project stack

1. [alchemypdf.db](https://github.com/onlineproducthouse/alchemypdf.db)
2. [alchemypdf.api](https://github.com/onlineproducthouse/alchemypdf.api)
3. [alchemypdf.api.httputils](https://github.com/onlineproducthouse/alchemypdf.api.httputils)
4. [alchemypdf.proc](https://github.com/onlineproducthouse/alchemypdf.proc)

2. [oph.tf.env](https://github.com/onlineproducthouse/oph.tf.env)
3. [oph.tf.devops](https://github.com/onlineproducthouse/oph.tf.devops)
